# site-web-les_100
Aperçu Détaillé du Projet:

The The 100 Fan Hub est un projet de site web entièrement statique, conçu pour servir de portail d'information clair et esthétique pour les fans de la série télévisée The 100. Techniquement, le site est construit sur des bases solides de HTML5 pour la structure et utilise amplement le framework Bootstrap 5.3, enrichi d'un CSS personnalisé puissant pour définir un thème visuel unique, par défaut en mode sombre. Le contenu est organisé à travers trois pages principales : l'Accueil, une section dédiée aux Personnages, et une page Saisons/Diffusions qui utilise des composants Accordéon pour présenter les résumés et les détails de diffusion de façon compacte et navigable. Une fonctionnalité clé, gérée par un petit script JavaScript, permet aux utilisateurs de basculer instantanément entre le thème sombre et un thème clair, améliorant ainsi l'accessibilité. Ce projet est par nature léger et facile à déployer, ne nécessitant aucune installation de serveur ou de base de données, ce qui en fait un excellent exemple de site statique complet et bien structuré.

Informations Essentielles:

    -Technologies : HTML5, CSS (styles personnalisés) et Bootstrap 5.3.

    -Contenu : Page d'accueil, Page des Personnages et Page des Saisons/Diffusions.

    -Fonctionnalité : Bascule entre Mode Sombre (par défaut) et Mode Clair.

    -Nature du Projet : Statique, ne nécessite ni serveur ni base de données.

Démarrage et Utilisation:

    -Télécharger : Récupérez l'ensemble des fichiers du projet.

    -Lancer : Ouvrez le fichier Index.html directement dans votre navigateur web.

   - Naviguer : Le site est immédiatement fonctionnel en local.
